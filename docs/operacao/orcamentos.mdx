# Orcamentos

## Resumo

O modulo de orcamentos deve ser ensinado como um fluxo pratico, desde a criacao ate a aprovacao.

## Passo a passo recomendado

1. escolher o cliente
2. adicionar produtos e servicos
3. revisar valores
4. aplicar desconto, se necessario
5. salvar e enviar

## Dicas

- mantenha exemplos simples
- explique cada status do orcamento

## Erros comuns

- esquecer itens obrigatorios
- aprovar sem revisar valores

## FAQ

### Posso transformar um orcamento em ordem de servico?

Sim. Essa deve ser uma das principais paginas desta secao.
