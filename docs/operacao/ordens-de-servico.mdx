# Ordens de servico

## Resumo

Esta e uma das areas mais importantes para oficinas e empresas de servico.

## Passo a passo recomendado

1. abrir a ordem
2. vincular cliente
3. registrar servicos
4. atualizar status
5. finalizar atendimento

## Dicas

- use nomes de status simples
- mantenha o fluxo visual em ordem cronologica

## Erros comuns

- esquecer de atualizar o andamento
- finalizar a ordem antes da revisao final
