# Notas fiscais

## Resumo

O conteudo fiscal deve ser direto, confiavel e sem excesso de jargao.

## Subcategorias recomendadas

- emitir nota
- revisar dados antes da emissao
- entender rejeicoes
- acompanhar status

## Erros comuns

- dados fiscais incompletos
- tentativa de emissao com cadastro incorreto

## FAQ

### O que revisar antes de emitir?

Cliente, servico, valores e dados fiscais obrigatorios.
