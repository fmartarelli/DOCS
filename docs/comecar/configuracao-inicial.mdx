# Configuracao inicial

## Resumo

Antes de usar os modulos principais, configure os dados basicos da empresa e os acessos dos usuarios.

## Passo a passo

1. Revise os dados da empresa
2. Confirme informacoes fiscais
3. Cadastre usuarios principais
4. Defina permissoes iniciais
5. Valide preferencias operacionais

## Dicas

- registre apenas os usuarios que realmente vao operar o sistema
- mantenha perfis de acesso simples no inicio

## Erros comuns

- liberar acesso total para todos os usuarios
- esquecer dados obrigatorios da empresa

## FAQ

### Posso ajustar isso depois?

Sim, mas o ideal e começar com o basico correto.
