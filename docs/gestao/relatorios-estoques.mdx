# Relatorios de Estoques

## Quando utilizar

- Quando quiser consultar o saldo de estoque.
- Quando precisar ver produtos com estoque positivo, zero ou negativo.
- Quando quiser comparar valor de custo e valor de venda em estoque.

## Como acessar

1. Entre em `Relatorios`.
2. Abra `Estoques`.

## O que a tela faz

Essa tela mostra os produtos e servicos ligados ao estoque selecionado, com seus saldos e valores.

## Filtros principais

- `Estoque`
- `Situacao do Estoque`
- `Situacao do Produto`
- `Tipo do Produto`

Opcoes encontradas:

- situacao do estoque
- `Todos`
- `Estoque positivo`
- `Estoque zero`
- `Estoque negativo`

- situacao do produto
- `Todos`
- `Ativo para venda`
- `Inativo para venda`

- tipo do produto
- `Todos`
- `Apenas produtos`
- `Apenas servicos`

## Informacoes mostradas

- `Produto`
- `Tipo`
- `Estoque`
- `Valor de Custo em Estoque`
- `Valor de Venda em Estoque`

## Recursos encontrados

- atualizacao por filtro
- tabela com totais
- ordenacao por coluna
- paginacao

Observacao importante:

- o botao visual de impressao esta comentado no HTML atual, embora exista metodo de geracao no componente

## Mensagens encontradas

- `Nenhum estoque encontrado nos filtros atuais`
- `Nao conseguimos recuperar os dados`

## Dicas importantes

- Use o filtro `Estoque negativo` para localizar possiveis problemas de saldo.
- Use `Apenas servicos` ou `Apenas produtos` para separar melhor a consulta.
- Compare valor de custo e valor de venda para entender o peso do estoque armazenado.
